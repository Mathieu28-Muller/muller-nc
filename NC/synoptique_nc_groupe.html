Synoptique — Module NC Parent-Satellites

Muller Automotive · ISO 9001:2015 · LNE · v1 — 05/05/2026

Vue d'ensemble — Flux NC parent-satellites

Plusieurs personnes de secteurs différents signalent le même problème → le système les regroupe sous une NC parent unique. L'analyse est faite une seule fois. La clôture est en cascade.

Création du groupe

Admin sélectionne ≥ 2 NC → saisit le libellé du groupe et le motif → valide.
Les NC passent en statut rattachee. Journal d'audit créé.

§ 10.2.2 LNE-4

Analyse unique

5P + Ishikawa + CAPA portés par la NC parent. Les satellites pointent vers l'analyse via badge cliquable → NC-P-XXXXX.

§ 10.2.1 LNE-1 LNE-2

Clôture en cascade

Admin saisit la preuve d'efficacité (min. 30 car.) → clôture simultanée de tous les satellites → notification individuelle de chaque déclarant.

§ 10.2.1e LNE-2 bloquant

Scénario 1 — NC décalée dans le temps (rattachement différé)

Un groupe est créé le 15 mars. Le 28 mars, un technicien SAV au Maroc signale NC-007 sur le même défaut. Le groupe est toujours ouvert.

Règles spécifiques

  • Aucune limite de délai pour rattacher à un groupe ouvert
  • La NC tardive enrichit l'analyse — elle ne l'invalide pas
  • Seuil d'alerte automatique : score ≥ 60/100
  • Motif tracé : [RATTACHEMENT TARDIF] dans le journal
  • Notification au déclarant dès le rattachement

Grille de score /100

Scénario 2 — NC après clôture parent (résiduel ou récidive)

Le groupe NC-P-001 est clôturé le 20 avril. Le 5 mai, un client signale le même défaut sur du matériel expédié en mars. La date de fabrication du matériel est la clé de décision.

CAS A — Résiduel §8.7

Matériel fabriqué avant la date de correction. Le défaut était déjà en circulation lors de la CAPA. La CAPA est valide et efficace.

Action : Créer un groupe résiduel lié au groupe clôturé (est_residuel=true, parent_clos_reference). CAPA propres : rappel, remplacement terrain, surveillance.

§ 8.7

CAS B — Récidive §10.2.1d

Matériel fabriqué après la correction. Le défaut réapparaît sur de la production post-CAPA. La CAPA est inefficace.

Action : Réouvrir le groupe. LNE-2 invalidée et archivée. Nouvelle analyse 5P obligatoire depuis la cause racine. Motif ≥ 20 caractères.

§ 10.2.1d LNE-2 invalidée

// Créer un groupe
NcParent.creerGroupe({
  label: 'Défaut boîtier lot F24-09',
  nc_ids: ['NC-001', 'NC-002', 'NC-003'],
  motif: 'Même symptôme SAV + Prod',
  admin_email: 'admin@muller.fr'
});

// Clôturer en cascade
NcParent.cloturerGroupe({
  parent_id: 'NC-P-00001',
  preuve: 'Aucune récidive sur 3 mois, stats jointes',
  admin_email: 'admin@muller.fr'
});

// Données
NcParent.getGroupes()         // [] tous les groupes
NcParent.getSatellites(pid)   // [] NC d'un groupe
NcParent.getJournal()         // [] journal immuable

Modèle de données — Nouvelles colonnes (si migration vers SQL)

-- Table nc_groupes (nouvelle)
CREATE TABLE nc_groupes (
  id                    VARCHAR(15) PRIMARY KEY,  -- 'NC-P-00001'
  groupe_label          VARCHAR(200) NOT NULL,
  gravite               ENUM('mineure','majeure','critique'),
  statut                ENUM('en_traitement','close','en_traitement_reouvert'),
  groupe_created_at     DATETIME NOT NULL,
  groupe_created_by     VARCHAR(100),
  groupe_motif          TEXT,
  est_residuel          BOOLEAN DEFAULT FALSE,
  parent_clos_reference VARCHAR(15),              -- FK → nc_groupes.id
  preuve_efficacite     TEXT,
  clos_le               DATETIME,
  clos_par              VARCHAR(100),
  reouvert_le           DATETIME,
  preuve_anterieure     JSON                      -- LNE-2 archivée
);

-- Table nc_satellites (relation NC ↔ groupe)
CREATE TABLE nc_satellites (
  nc_numero             VARCHAR(20) NOT NULL,
  parent_id             VARCHAR(15) NOT NULL,     -- FK → nc_groupes.id
  rattachement_date     DATETIME NOT NULL,
  rattachement_by       VARCHAR(100),
  rattachement_motif    TEXT,
  cloture_via_parent    VARCHAR(15),
  PRIMARY KEY (nc_numero, parent_id)
);

-- Table nc_journal_groupes (audit immuable)
CREATE TABLE nc_journal_groupes (
  id          INT AUTO_INCREMENT PRIMARY KEY,
  timestamp   DATETIME NOT NULL,
  action      VARCHAR(50) NOT NULL,
  parent_id   VARCHAR(15),
  nc_id       VARCHAR(20),
  by_user     VARCHAR(100),
  motif       TEXT,
  extra_json  JSON                                -- données variables
);

async function creerGroupeRemote(groupeData) {
  const res = await fetch('/api/nc-groupes', {
    method: 'POST',
    headers: { ...authHdr() },
    body: JSON.stringify(groupeData)
  });
  if (!res.ok) throw new Error(await res.text());
  return await res.json();
}

  • Réouverture sans motif ≥ 20 caractères → § 10.2.1d NC mineure
